🎵 Musify – Spotify-Inspired Web Application

by Othmane Settat . Ryan Bajjou · Alaa Maachouk 

📌 Project Overview

Musify is a music streaming web application inspired by Spotify.

Initially developed as a static website using HTML, CSS, and JavaScript, the project is now evolving into a dynamic full-stack application using PHP and MySQL.

The goal is simple:

Create a platform where users can explore music, interact with content, and enjoy a personalized experience — similar to real streaming services, but in a simplified academic version.
